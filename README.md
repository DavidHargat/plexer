# plexer

`plexer` is a lil C http server created for fun :)

## usage

```
plexer [-hv] [-p port]
```

`plexer` always serves your current directory.

example:

`./plexer -p 8080` run a server on port 8080

## build / run

```
clone https://github.com/af-inet/plexer

cd plexer

make

./plexer
```
